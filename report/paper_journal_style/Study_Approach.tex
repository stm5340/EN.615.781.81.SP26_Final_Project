\section{Study Approach} \label{sec:study-approach}

In this section we describe the methodological framework used to study the BB84 quantum key distribution protocol. Our implementation is organized in several logical blocks: (i) the simulation framework and backends, (ii) baseline BB84 state preparation, measurement, and sifting on simulators and hardware, (iii) an intercept--resend eavesdropper model, and (iv) classical post--processing, including error correction and privacy amplification. 
A Qiskit implementation of this framework can be found at \cite{bb84_project_2026}.
